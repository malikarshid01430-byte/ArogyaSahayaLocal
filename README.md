🏥 ArogyaSahayaLocal – Android App

A modern Android healthcare application built to manage patient records, doctor details, and medical data in a structured and efficient way. It helps you store, update, and track health information locally with a fast and reliable system.

🚀 Features

• 🔐 Secure login and logout system
• 👤 Patient profile creation and updates
• 🧑‍⚕️ Doctor list with names and contact numbers
• 📅 Appointment and next checkup tracking
• 💊 Medicine schedule and reminders
• 🔔 Notification system for checkups and medicines
• 📊 Professional dashboard with health overview
• 🗂️ Add, edit, and delete medical records
• 🔍 Search and filter patient data
• 🌐 Multi language support including Indian languages
• ⚙️ Settings and profile update module
• 💾 Offline data storage using local database
• ⚡ Fast, smooth, and responsive performance

🛠️ Tech Stack

• Language: Java or Kotlin
• Platform: Android Studio
• Database: SQLite or Room Database
• UI: Material Design components
• Architecture: MVVM

📂 Project Structure

com.example.arogyasahayalocal

ui/ → Activities and UI logic
data/ → Models and database handling
adapter/ → RecyclerView adapters
utils/ → Helper and utility classes

🎯 Purpose

This app helps you manage healthcare data in one place. You can store patient records, track medical history, manage doctor contacts, and monitor treatments without internet dependency.
